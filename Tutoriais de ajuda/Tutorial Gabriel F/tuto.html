Tutorial

1º Passo:

Cadastrar usuario

Para cadastrar um usuario é precisso informar alguns dados:
ParaExemplo:
- Usuario: Gabriel
- Senha: senha123
- Email: bielsilvabh@mysqltools.com.br



Para confirmar o cadastro, basta clicar 1 vez no botão cadastrar.

2º Passo:


Recupar senha
Para recuperar um usuario cadastrado é precisso informar alguns dados:
Exemplo:
- Usuario: Gabriel
- Email: bielsilvabh@mysqltools.com.br



Para recuperar a senha basta apertar o botão confirmar, e sua nova senha será enviada para o email informado.
