  • O que é um BANCO DE DADOS(BD):

    • Um BANCO DE DADOS é uma importante base de dados criada para armazenar dados que se relacionam algum jeito criando algum sentido. Normalmente permanecem por anos até serem alterados, pois são criados de acordo co a nessessidade de cada sistema e só é alterado quando as necessidades do cliente mudam. Existem Vários bd, alguns deles são o MySQL e o Oracle, mas a diferença maior entre eles é a velocidade de resposta.

  • Para que serve o banco de dados:

    • É usado por quase todos os sistemas e é ele que armazena os dados que o usuario nessessita para uso posterior do sistema. Por exemplo: o repositor de um mercado faz uma alteração no estoque, adicionando quantidade de algum produto, essa quantidade adicionada vai para o bd e sofre baixa sepre que for vendido unidades daquele produto. Também armazena informações de clientes e contatos, materias primas e etc...

  • O que é uma Tabela:

    • Tabela é um conjunto de dados que devem ser armazenados juntos para manter uma lógica no uso do banco de dados. Exemplo: Para uma loja cadastrar seus clientes, guardar valores como o nome e o cpf do cliente.

  • O que é uma Primary Key:

    • Primary Keys, é a chave primaria de uma tabela de um BD, ela é como a identidade de cada registro da tabela, pois não pode repetir, na maioria da vezes é codigo e auto incrementado pelo banco de dados. É o melhor jeito para se referir a algum registro específico.

  • O que é uma Foreign Key:

    • Foreign Key é o jeito de agrupar o registro de uma tabela a outra, relacionando eles. Por exemplo: no cadastro do cliente, é cadastrado apenas um nome e cpf, mas pode ser necessário mais de um numero de telefone, esses numeros vão para uma outra tabela e cada registro de um telefone novo tem a Foreign Key com o valor da Primary Key da tabela do Cliente.
